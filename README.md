# iptv-jh

自动整理的 IPTV 播放列表仓库。

## 文件

- `iptv.m3u`：主播放列表

## 直接订阅链接

```text
https://raw.githubusercontent.com/michaelactions/iptv-jh/main/iptv.m3u
```

## 使用方法

把上面的链接导入支持 M3U 的播放器即可，例如：

- VLC
- TiviMate
- Kodi
- IPTV Smarters
- 各类电视盒子 / 手机 IPTV 播放器

## 说明

- 播放源来自公开网络收集
- 可用性会随时间变化
- 仓库只保存播放列表，不存储视频内容
