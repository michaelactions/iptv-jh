# iptv-jh

自动整理的 IPTV 播放列表仓库。

## 文件

- `iptv.m3u`：主播放列表（每日更新）

## 订阅链接

### 直接链接（GitHub）

```text
https://raw.githubusercontent.com/michaelactions/iptv-jh/main/iptv.m3u
```

### 国内加速链接

#### Cloudflare 代理



## 使用方法

把上面的链接导入支持 M3U 的播放器即可，例如：

- **电脑端**: PotPlayer, VLC
- **安卓/TV 端**: TiviMate, DIYP 影音，Televizo
- **iOS 端**: APTV, zFuse
- **电视盒子/手机**: 各类 IPTV 播放器

## 频道分类

播放列表包含以下频道分类：

- **央视频道**: CCTV 系列、CGTN、央视付费频道等（约 40+ 频道）
- **卫视频道**: 各省卫视（约 40+ 频道）
- **地方频道**: 各地市级电视台（约 340+ 频道）

总计约 **430+** 个电视频道。

## 说明

- 播放源来自公开网络收集，不定期更新维护
- 可用性会随时间变化，如遇无法播放请切换其他源或等待更新
- 仓库只保存播放列表，不存储视频内容
- 仅供学习研究使用，禁止商业用途

## 推荐播放器

| 平台 | 推荐软件 | 特点 |
|------|---------|------|
| 安卓 TV | TiviMate | 自动刷新，解码强，支持 EPG 和回看 |
| iOS | APTV | 支持 EPG 和回看，界面美观 |
| 电脑 | PotPlayer | 功能强大，支持多种格式 |
| 跨平台 | VLC | 开源免费，兼容性好 |

## 注意事项

1. 部分频道可能需要 IPv6 网络才能播放
2. 如遇卡顿可尝试切换其他频道的备用源
3. 建议开启播放器的"自动刷新"功能
4. 定期更新订阅链接以获取最新频道
